# Blog-API
API for Blog CRUD

## Instalation

### Setup Environment
```python3 -m virtualenv .env```

```source .env/bin/activate```

```git clone https://github.com/InersIn/Blog-API.git```

```cd Blog-API```

```pip3 install -r ./requirements.txt```

### Import Database

```mysql -u admin -p Blog < database.sql``` // Make sure database Blog is exists

### Run Application
```./manage.py runserver```
